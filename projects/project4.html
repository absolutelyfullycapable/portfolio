사내 업무로 센서융합보안기술의 선두 기업인 '주식회사 세오'의 공식 웹사이트 리뉴얼 프로젝트에 투입되어 퍼블리싱을 전담하였습니다.

웹사이트 각 섹션의 HTML 파일은 해당 섹션 별 폴더에 넣어 정리하고, CSS 파일은 섹션 별 하나로 통합하여 더 쉽게 유지 보수가 가능하도록 구축하였습니다.

또, 퍼블리싱과 함께 디자인 보조 업무를 도우며 웹 디자이너와 소통하고 더 나은 방향을 찾기 위해 노력하였습니다. 그 덕분에 퍼블리싱을 비롯한 업무의 전반적인 프로세스를 익힐 수 있었습니다.

홀로 웹 퍼블리싱 파트를 진행하는 것이 쉽지 않았지만, 스스로 작성한 퍼블리싱 가이드를 도입하여 체계적으로 진행하고 마무리할 수 있었습니다.
