현대자동차(주)

2024

Overview

HMG Driving Experience는 현대자동차그룹이 운영하는 체험형 드라이빙 프로그램으로, 고객이 다양한 차량을 직접 주행하며 성능과 브랜드를 경험할 수 있도록 하는 공식 예약 플랫폼입니다.

기존 소스를 유지하는 동시에 UI 개선이 요구되는 프로젝트였습니다. 리스크를 줄이고자 신규 작업은 별도의 파일로 분리하여 우선 적용하는 방식으로 진행했습니다.

그 결과, 기존의 안정성을 해치지 않으면서 사용자 경험을 개선할 수 있었습니다.

기존 소스 기반의 스타일 및 구조 개선

기존 소스를 기반으로 UI 개선이 요구되는 프로젝트였습니다. 이에 기존 소스 수정 없이 신규 소스가 우선 적용되도록 작업하여 영향을 최소화하면서도 안정적으로 UI가 개선될 수 있도록 작업했습니다.

사용자 입력 및 상태에 따른 동적 UI 개선

마이페이지 및 예약하기 페이지는 사용자의 선택 및 입력 값에 따라 다양한 UI 상태가 발생하는 구조였습니다.

큰 틀에서 공통적인 UI를 먼저 구성한 다음, 상태에 따라 달라지는 영역은 기능 단위로 분리 후 작업하여 복잡한 화면 구성을 체계적으로 정리할 수 있었습니다.
