안녕하세요, 웹 퍼블리셔 이지혜입니다! 👋🏻

웹 퍼블리싱을 기반으로 웹사이트 구축부터 운영까지 경험하며, 사용자 경험을 고려한 UI 구현을 지향합니다.

단순한 페이지 구현을 넘어 공통 컴포넌트 구조와 UI 가이드를 선행 설계하고, 이후 운영 단계에서도 일관성과 유지보수 효율을 높일 수 있는 퍼블리싱에 강점을 가지고 있습니다.
또 다양한 운영 프로젝트를 통해 웹 접근성 관리와 기능 개선, 유지보수 작업에 참여하며 실무 전반에 대한 이해를 쌓아왔습니다.

맡은 프로젝트의 목적을 우선적으로 고려하며, 기획자·디자이너·개발자와의 원활한 협업을 중요하게 생각합니다.
이를 통해 사용자와 서비스 모두에게 더 나은 경험을 제공하는 UI를 만들고자 합니다.
